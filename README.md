

## Setup

### Prerequisites
- Node.js installed


### Installation
1. Clone the repository:
   ```sh
   git clone <repo_url>
   cd <project_directory>
   ```
2. Install dependencies:
   ```sh
   npm install
   ```

## Running the App

### Using Nodemon (Development)
```sh
npm run dev
```


### Environment Variables
Create a `.env` file in the project root with:
```
PORT
```
